# Scratchpad

**基于electron的javascript代码草稿纸，可运行js代码并用console.log、log、alert输出执行结果**

- ctrl+r运行
- ctrl+l清屏
- 支持代码高亮
- 右下角行列信息
- 输出结果色彩区分
- 点击输出结果“行号”跳到对应的源代码
